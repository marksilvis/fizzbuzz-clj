# fizzbuzz-clj

FizzBuzz in clojure

## Installation

Download from https://github.com/marksilvis/fizzbuzz-clj/

## Run

Run with leiningen

    $ lein run [value]

## Build

Build with leiningen

    $ lein uberjar

## Usage

After building

    $ java -jar fizzbuzz-clj-0.1.0-standalone.jar [value]
